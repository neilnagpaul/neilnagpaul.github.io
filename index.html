Projects

  • Kind Together

    Kind Together is a project that I am leading to help connect volunteers with people in need. Unfortunately, I cannot say much more about until the project is officially launched. Stay tuned!

  • Hands to Words

    Hands to Words is a science fair project that I am currently working on with a few friends. The goal of the project is to create a device that can translate American Sign Language (ASL) into English text. The device uses a camera to capture hand gestures and a machine learning model to interpret the gestures. The project is still in the early stages of development, but we hope to have a working prototype soon!

  • Garagebot

    Garagebot was a project that used the myQ API to control a garage door opener to avoid paying steep fees for a smart garage door opener. You can read more about it in my blog post.

  • Photo Narrations

    I built a website from scratch for a local wedding photographer. The website showcases the photographer's work and allows users to get in touch. However, the website is no longer live.
